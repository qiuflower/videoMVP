# 🎥 视频重绘生成片段存放目录 (Generated Redraw Videos Directory)

此目录用于存放您通过 Runway, Luma, Kling 等工具重绘生成的新镜头视频片段。

## 命名规范与建议

为保证分镜剧本超链接和最终拼接流程的顺利匹配，建议将各镜重绘后的视频重命名为对应的镜号：
- 第 1 镜：`Scene-001_redraw.mp4`
- 第 2 镜：`Scene-002_redraw.mp4`
- ...
- 第 38 镜：`Scene-038_redraw.mp4`

## 为什么这样做？

在最终的 `storyboard.md` 分镜表中，`相关链接` 列会自动生成指向该目录下对应文件名称的超链接。将视频放于此目录下可以方便在分镜表中直接点击播放、对比与审核。
